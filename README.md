# Airbnb Price Regression using SEMMA

## Overview  
This notebook follows the SEMMA process (Sample, Explore, Modify, Model, Assess) to develop a regression model predicting nightly Airbnb listing prices. Using an Inside Airbnb dataset from Singapore, the analysis focuses on modeling log-transformed prices based on property and location features.

## Features  
- Data sampling and initial exploration of Airbnb listings  
- Handling of missing values and outliers with clear data cleaning steps  
- Detailed column-level documentation and variable treatment decisions  
- Creation of derived variables such as log-transformed price to improve model fit  
- Extensive visual exploratory data analysis with interpretative commentaries  
- Linear regression modeling using relevant predictors  
- Model evaluation using metrics like RMSE, MAE, and R²  

## Tech Stack  
- Python  
- Jupyter Notebook / Google Colab compatible  
- Libraries: pandas, numpy, matplotlib, scikit-learn  

## How to Use  
1. Clone or download the repository.  
2. Upload the required data files (`listings.csv`, `neighbourhoods.csv`, `reviews.csv`) to the working folder or Colab environment.  
3. Open and run the notebook `airbnb-price-regression-semma.ipynb` step-by-step to reproduce the analysis and results.
